Official logo files for LayerDrop3D - 3D Print-on-Demand Service

## Icon Versions (Square)

### Blue (Primary Brand Color)
- `logo-icon-blue-transparent.svg` - Blue icon on transparent background (#00A8FF)

### White
- `logo-icon-white-transparent.svg` - White icon on transparent background

### Black
- `logo-icon-black-transparent.svg` - Black icon on transparent background

## Full Logo Versions (with Text)

### Blue (Primary Brand Color)
- `logo-full-blue-transparent.svg` - Full logo with text on transparent background (#00A8FF)

### White
- `logo-full-white-transparent.svg` - Full logo with text on transparent background

### Black
- `logo-full-black-transparent.svg` - Full logo with text on transparent background

## Usage

- **Social Media Profiles**: Use icon versions (square format)
- **Website Header/Footer**: Use full logo versions
- **Favicon**: Use blue icon version, scaled to 16x16 or 32x32px
- **Dark Backgrounds**: Use white or blue icon
- **Light Backgrounds**: Use black or blue icon
- **Print/Merch**: Use full logo versions with padding

## Color Palette

- **Primary Blue**: #00A8FF
- **Dark Background**: #0f1419
- **Text**: White (#FFFFFF) on dark, Black (#000000) on light

## Technical Notes

All logos are provided as SVG files for scalability. They can be:
- Used directly in web projects
- Exported to PNG, PDF, or other formats
- Resized without quality loss
- Imported into design tools
