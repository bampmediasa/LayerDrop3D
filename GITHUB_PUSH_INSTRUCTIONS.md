# Push to GitHub - Simple Instructions

## Option A: Manual (2 minutes)

1. Open Terminal and run:
```bash
git clone https://github.com/bampmediasa/LayerDrop3D.git
cd LayerDrop3D
mkdir -p website assets/logos
```

2. Copy these files from BAMP Prints folder into the LayerDrop3D folder:
   - website/index.html
   - website/products.html
   - website/logo-icon-blue.png
   - website logo SVG files (from /assets/logos)

3. Run:
```bash
git add .
git commit -m "Add website and branding assets"
git push
```

## Option B: Using GitHub Desktop

1. Go to https://github.com/bampmediasa/LayerDrop3D
2. Click "Code" → "Open with GitHub Desktop"
3. Clone to your computer
4. Copy the files from this folder into the cloned repo
5. Commit and push from GitHub Desktop
